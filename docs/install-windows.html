Windows Installation Guide

Windows SmartScreen may block the installer.
Here's how to run Lumen AI.

You only need to do this once.

1 Download the .exe file from the release page
2 Run the installer — Windows SmartScreen will block it. Click More info
3 Click Run anyway
4 Choose installation options and click Next
5 Choose install location and click Install
6 Wait for installation to complete
7 Click Finish to launch Lumen AI
