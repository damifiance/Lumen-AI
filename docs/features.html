Every feature, at a glance

Everything Lumen AI can do to make your research workflow faster and smarter.

📄

PDF Reading

A modern reader built for academic papers

Clean PDF Viewer

Read papers in a distraction-free interface powered by PDF.js. Pages scale to fit your window width automatically.

Multi-Tab Support

Open up to 9 papers simultaneously. Switch between them instantly with keyboard shortcuts (Option/Alt + 1-9).

Close & Manage Tabs

Close individual tabs with Option/Alt + W. Click any tab header to switch. Paper titles and page counts shown at a glance.

Folder Browser

Navigate your filesystem, pin your papers folder, and jump to Desktop, Documents, or Downloads with one click.

🎨

Highlighting

Mark up papers in multiple colors, saved automatically

5 Highlight Colors

Choose from Yellow, Green, Blue, Pink, and Purple. Select text and pick a color from the floating toolbar.

Auto-Saved

Every highlight is instantly persisted to a local SQLite database. Your work is never lost.

Recolor & Delete

Change highlight colors after creation. Remove any highlight with one click from the hover popup.

Ghost Preview

See a preview of the highlight before committing. Hover over existing highlights to see a popup with actions.

📝

Notes & Annotations

Attach notes to any passage, organized in threads

Threaded Notes

Create multiple notes on the same text selection. Each note is timestamped and shown in a threaded view.

Edit & Delete

Edit any individual note inline, or delete specific notes from a thread. Full control over your annotations.

Dashed Underline Display

Notes appear as dashed underlines in the PDF, distinct from color highlights, so you can spot them at a glance.

Save AI Answers as Notes

When AI explains a passage, save that answer directly as a note attached to the highlighted text.

✨

AI Chat

Ask questions about any passage or the entire paper

Select & Ask

Selection-Based Questions

Select any text, type a question, and get an instant AI explanation about that specific passage.

Full Paper

Full-Paper Chat

Open the chat panel (Option/Alt + /) and discuss the entire paper. Multi-turn conversations with full context.

Streaming

Real-Time Streaming

Responses stream token-by-token with a live animation. Stop generation mid-stream if you have enough.

Multi-Model Support

Switch between Ollama (local, free), OpenAI GPT-4o, and Anthropic Claude from a dropdown. Your preferred model is remembered.

Markdown & LaTeX

AI responses render full Markdown with KaTeX math formulas and syntax-highlighted code blocks.

Clear & Control

Clear chat history with one button. Stop streaming anytime. Chat panel shows which paper is being discussed.

⌨

Keyboard Shortcuts

Fully customizable shortcuts for power users

Customizable

Remap Any Shortcut

Press Option/Alt + K to open the shortcut editor. Click any shortcut and record a new key combination.

12+ Default Shortcuts

Switch tabs (1-9), close tabs (W), toggle chat (/), open shortcut editor (K), send messages (Enter), and more.

Cross-Platform Keys

Automatically handles Option on macOS and Alt on Windows/Linux. Physical key detection works across keyboard layouts.

Reset to Defaults

Customizations saved to localStorage. One-click reset to restore all default shortcuts.

🔒

Privacy & Data

Your papers and notes never leave your machine

Local First

Runs 100% Locally

With Ollama, everything runs on your machine. No cloud, no subscriptions, no data leaving your computer.

SQLite Database

Highlights, notes, and paper metadata stored in a local SQLite database. Fast, reliable, and portable.

No Account Required

No sign-up, no login, no tracking. Install and start reading immediately.

Optional Cloud AI

Add OpenAI or Anthropic API keys only if you want cloud models. Completely optional — Ollama works out of the box.

💫

UX & Polish

Small details that make a big difference

Guided Onboarding

5-step welcome walkthrough on first launch. Skip it or re-open anytime from the sidebar help button.

Pin Your Folder

Pin your papers folder so it opens automatically every time you launch the app. One less click.

Floating Action Menu

Select text and a floating toolbar appears with highlight colors, note, and Ask AI buttons. No right-clicking needed.

Loading States

Smooth loading spinners for PDFs, streaming animations for AI responses, and visual feedback throughout.

💻

Cross-Platform

Desktop apps for macOS, Windows, and Linux

macOS (Apple Silicon)

Native .dmg installer for Apple Silicon Macs. Drag to Applications and launch.

Windows

NSIS installer with custom install directory, desktop shortcut, and auto-launch after install.

Linux

AppImage — no installation needed. Make executable and run on any Linux distribution.

Web Version

Run from source in any browser at localhost:5173. Same full experience without the desktop wrapper.
